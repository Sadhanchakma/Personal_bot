🤖 Premium Subscription Telegram Bot

💎 A simple and powerful Telegram bot to sell premium services like Netflix, Spotify, etc.

---

🚀 Features

- 💰 User Balance System
- ➕ Add Balance via bKash / Nagad / Rocket / Binance
- 📸 Screenshot Verification System
- 🛒 Buy Services (Netflix, Spotify)
- 📦 Plan Selection (Share / Private)
- 🔐 Admin Balance Control
- 📲 Clean & User-Friendly UI

---

🛠️ Tech Used

- Python 🐍
- python-telegram-bot
- JSON Database

---

📂 Project Structure

📁 Personal_bot
 ├── main.py        # Bot runner
 ├── handlers.py    # All bot logic
 ├── database.py    # Data handling
 ├── config.py      # Token & Admin ID

---

⚙️ Setup

1️⃣ Clone the repo:

git clone https://github.com/yourusername/yourrepo.git
cd yourrepo

2️⃣ Install requirements:

pip install python-telegram-bot

3️⃣ Edit "config.py":

TOKEN = "YOUR_BOT_TOKEN"
ADMIN_ID = YOUR_TELEGRAM_ID

4️⃣ Run the bot:

python main.py

---

👨‍💻 Admin Command

/addbalance user_id amount

Example:

/addbalance 123456789 500
